# Handpicked

A lean-back, TV-like experience built from curated YouTube videos.

## What this is

- A linear, channel-based viewing experience
- Human-curated playlists presented as TV channels
- Designed for passive, background watching
- No viewer accounts or interaction

## Tech Stack

- Frontend: Nuxt 3 / Vue 3
- Backend: Supabase (Postgres, Auth)
- Hosting: Vercel
- Video: YouTube IFrame API

## Local development

```bash
cd app
pnpm install
pnpm dev
